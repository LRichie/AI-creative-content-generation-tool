# AI创意文案生成工具

## 项目简介

AI创意文案生成工具是一款基于先进人工智能技术的文案创作辅助系统，旨在帮助用户快速生成高质量、多风格的创意文案。该工具集成了图像分析、文案生成和合规检查三大核心功能模块，通过LangGraph工作流引擎实现自动化文案创作流程，为营销、广告、内容创作等领域提供高效解决方案。

## 功能特性

### 图像分析

自动识别图像内容特征，提取关键视觉元素，为文案创作提供精准的视觉语义支持，实现图文内容的深度融合。

### 文案生成

基于先进的自然语言处理模型，支持多种文案风格（如营销、广告、社交媒体等）的自动生成，可根据用户需求进行个性化调整。

### 合规检查

内置多维度合规检测机制，自动识别文案中的潜在风险，确保生成内容符合相关法律法规和平台规范，降低合规风险。

## 技术架构

### 技术栈

- **后端框架**：FastAPI- **工作流引擎**：LangGraph- **数据库**：PostgreSQL- **容器化**：Docker
### 系统架构

系统采用模块化设计，主要分为图像分析模块、文案生成模块、合规检查模块和API服务层，通过LangGraph实现各模块间的流程编排和数据流转，确保系统高效稳定运行。各模块通过RESTful API通信，PostgreSQL存储文案模板与用户配置，LangGraph负责状态管理与错误重试

## 安装部署步骤

### 环境要求

- Docker- Docker Compose- Git
### 部署步骤

- 克隆项目仓库
```bash
git clone https://github.com/yourusername/ai-creative-copywriter  # 请替换为实际仓库地址
cd AI-creative-content-generation-tool

```

- 启动数据库服务
```bash
cd ./docker/postgresql
docker-compose up -d

```

- 启动后端服务
```bash
cd ..
python backendServer

```
